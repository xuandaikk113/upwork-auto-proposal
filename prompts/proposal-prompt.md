## Proven Template Structure
Use this format that has generated $500K+ in Upwork revenue:

```
Hi [name if mentioned], I do {relevant_thing} all the time.
So confident I'm the right fit that I created a solution diagram + proposal for you: $$$

About me: {1-2 sentences of relevant background with specific achievements}

{1-2 sentences showing you understand their specific problem and how you'd solve it}

Happy to discuss details—just respond to this proposal.

Thank you!
```

## Critical Rules
1. $$$ is a placeholder for the Google Doc link - leave it EXACTLY as $$$
2. Casual, spartan tone - NO emojis, NO flowery language
3. Reference SPECIFIC details from the job description
4. Keep it under 150 words total
5. If a name appears in the job description, use it after "Hi"
6. Focus on THEIR problem, not your resume
7. Use first-person "I" language

## Output
Return ONLY the proposal text, ready to copy and submit. Nothing else.
